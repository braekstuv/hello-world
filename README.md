# Installation and Usage

Clone the repository, cd into it and run the install script:

```sh
git clone https://github.com/braekstuv/hello-world.git
cd ./hello-world
sh ./install.sh
```

To use the cli, just run:

```sh
hello-world
```

If you wish to uninstall, just run

```sh
sh ./uninstall.sh
```
